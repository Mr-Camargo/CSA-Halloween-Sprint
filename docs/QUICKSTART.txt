║       🎃 HAUNTED HOUSE ADVENTURE - QUICK START GUIDE 👻           ║
╚════════════════════════════════════════════════════════════════════╝

┌─────────────────────────────────────────────────────────────────────┐
│ 🚀 TO RUN THE GAME                                                  │
└─────────────────────────────────────────────────────────────────────┘
  PowerShell Commands:
    javac -d bin Main.java src/*.java
    java -cp bin Main

┌─────────────────────────────────────────────────────────────────────┐
│ 🎯 OBJECTIVE                                                        │
└─────────────────────────────────────────────────────────────────────┘
  Navigate through the haunted mansion and reach the EXIT DOOR!
  Defeat monsters, collect items, and survive the night!

┌─────────────────────────────────────────────────────────────────────┐
│ 🗺️  ROOM LAYOUT                                                     │
└─────────────────────────────────────────────────────────────────────┘
  START: Entrance Hall
    ├─→ Dark Library ─→ Haunted Bedroom ─→ Dusty Attic ─→ EXIT ✓
    └─→ Kitchen ─→ Basement ─→ Dusty Attic ─→ EXIT ✓

┌─────────────────────────────────────────────────────────────────────┐
│ 👾 MONSTERS (in order of difficulty)                                │
└─────────────────────────────────────────────────────────────────────┘
  1. Skeleton    (Library/Basement)   - 40 HP, 15 STR, 20% miss
  2. Slime       (Basement)           - 40 HP, 5+ STR, RAMPING DMG!
  3. Zombie      (Kitchen)            - 100 HP, 10 STR, tank
  4. Ghost       (Bedroom)            - 30 HP, 10 STR, phases
  5. Vampire     (Attic - BOSS)       - 120 HP, 40 STR, summons bats

┌─────────────────────────────────────────────────────────────────────┐
│ 🎒 KEY ITEMS TO COLLECT                                             │
└─────────────────────────────────────────────────────────────────────┘
  MUST HAVE:
    • Rusty Sword (Entrance) - +10 strength permanently
    • Health Potions (multiple rooms) - heal 15 HP
  
  HELPFUL:
    • Magic Amulet (Bedroom) - +10 strength
    • Poison Vial (Basement) - 30 damage to monster
    • Ancient Scroll (Library) - 20 damage to monster
    • Holy Water (Attic) - 20 damage to monster

┌─────────────────────────────────────────────────────────────────────┐
│ ⚔️  COMBAT CONTROLS                                                 │
└─────────────────────────────────────────────────────────────────────┘
  1 - Attack monster
  2 - Use item (healing or offensive)
  3 - Flee to random exit

┌─────────────────────────────────────────────────────────────────────┐
│ 🎮 EXPLORATION CONTROLS                                             │
└─────────────────────────────────────────────────────────────────────┘
  1 - Move to another room
  2 - Pick up item
  3 - View inventory
  4 - View status (health/strength/location)
  5 - Look around (re-examine room)
  6 - Quit game

┌─────────────────────────────────────────────────────────────────────┐
│ 💡 PRO TIPS                                                         │
└─────────────────────────────────────────────────────────────────────┘
  ✓ Grab Rusty Sword FIRST from Entrance Hall
  ✓ Collect all health potions you find
  ✓ Kill Slime FAST before damage ramps up
  ✓ Save offensive items for Ghost and Vampire
  ✓ Fight Ghost when it's NOT phased
  ✓ Stock up before facing Vampire in Attic
  ✓ Fleeing is tactical - heal and return!

┌─────────────────────────────────────────────────────────────────────┐
│ 📊 STARTING STATS                                                   │
└─────────────────────────────────────────────────────────────────────┘
  Health: 100 HP (max)
  Strength: 10 damage per attack
  Inventory: Empty

┌─────────────────────────────────────────────────────────────────────┐
│ 🏆 RECOMMENDED PATH (EASIEST)                                       │
└─────────────────────────────────────────────────────────────────────┘
  1. Entrance Hall → Get Rusty Sword + Health Potion
  2. Dark Library → Fight Skeleton (easier with +10 STR)
  3. Haunted Bedroom → Fight Ghost, get Magic Amulet
  4. Dusty Attic → Get items, fight Vampire (now have +20 STR!)
  5. Exit Door → WIN! 🎉

┌─────────────────────────────────────────────────────────────────────┐
│ ⚠️  DIFFICULTY NOTES                                                │
└─────────────────────────────────────────────────────────────────────┘
  • Slime gets STRONGER each turn - priority kill!
  • Ghost phases and dodges - be patient
  • Vampire is TOUGH - bring health potions and items
  • Bats from Vampire add up - kill Vampire first
